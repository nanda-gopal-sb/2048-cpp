# 2048 Game in C++

## Installation
To compile and run the game, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/2048.git
    ```
2. Navigate to the project directory:
    ```sh
    cd 2048
    ```
3. Compile the code:
    ```sh
    g++ -o 2048 main.cpp
    ```
4. Run the game:
    ```sh
    ./2048
    ```
